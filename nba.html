Code School Project Ball

Create a new HTML page.

Give it a custom title, add 1 image, 1 text element, and add a custom CSS and JS file to the page

In your Javascript

Define an Object Constructor for a Basketball Player

They need the following properties:

  • Name,
  • Age,
  • Height,
  • Weight,
  • 2Pt Shooting Percentage,
  • 3Pt Shooting Percentage,
  • Free Throw Shooting Percentage

They also need the following methods:

  • Dribble,
  • Shoot(Parameter for Free Throw/2pt Field Goal/3pt attempt)
    • For the Shoot method, utilize the player percentages to influence your algorithm for shooting. A 35% shooter should be able to make any shot base on a random number that generates lower than 35 for example
  • Pass

Stretch Goals

  • Render a basketball in canvas
  • Animate or move the ball when the Player methods run
  • Tie the different Player Methods to HTML buttons
  • Add a score variable that increments based on successful shots
  • Display the score in the HTML
